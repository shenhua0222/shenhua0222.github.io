heroku部署app教程

Posted on 2017-05-07   |   In node

heroku平台

认识这个平台是偶然逛知乎知道的，了解到这个可以对node进行服务器渲染，当时就有个想法，要把自己的个人主页部署在heroku上，这样就不用自己掏钱买服务器了。

部署过程

  • 注册Heroku账户
  • 安装Heroku Toolbelt
  • 创建要部署的app
  • 链接数据库 （静态网页可省略此步骤）
  • 部署

注册

  1. 注册需要gmail邮箱，可以翻墙或在qq邮箱注册一个gmail邮箱；
  2. 利用gmial邮箱，在官网 注册heroku账号；

部署

  1. 登陆 

    C:\Users\Administrator.PC-20160621BLYH>heroku login
Enter your Heroku credentials:
Email: qq1448143934@gmail.com
Password: \************
Logged in as qq1448143934@gmail.com

  2. 在heroku 上创建app，以便接收你的代码

    $ heroku apps: create [name]

  3. 进入到本地app目录，推送代码到heroku上的app

    $ git push heroku master

    备注: 如果在heroku上已经创建了的app上推送内容，可用如下代码指定相关app，然后推送

    $ heroku git:clone -a [appName]
